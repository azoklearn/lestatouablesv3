# Règles Firebase à appliquer dans la console Firebase

## Règles Firestore (Database)
Allez dans Firebase Console > Firestore Database > Rules et remplacez par :

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Permettre la lecture et l'écriture pour toutes les collections
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

## Règles Storage
Allez dans Firebase Console > Storage > Rules et remplacez par :

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if true;
    }
  }
}

## Instructions :
1. Ouvrez https://console.firebase.google.com
2. Sélectionnez votre projet "lestatouables-43ef0"
3. Allez dans "Firestore Database" > "Rules"
4. Copiez-collez les règles Firestore ci-dessus
5. Cliquez sur "Publier"
6. Allez dans "Storage" > "Rules" 
7. Copiez-collez les règles Storage ci-dessus
8. Cliquez sur "Publier"

Ces règles permettront la suppression de toutes les données.
